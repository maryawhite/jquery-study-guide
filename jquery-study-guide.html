JQuery

  • JQuery makes it easy to:
    • Navigate a document
    • select DOM elements
    • create animations
    • handle events
    • create AJAX applications
    • add plugins
    • source: Codeup Curriculum

Using Jquery:

  • Include the JQuery CDN in your script tag.
  • The $ dollar sign is an alias for JQuery.
    • $("body").css("padding", "1em") is interchangeable with JQuery("body).css("padding", "1em")
  • Use the document ready event listener and put your code inside.
Document Ready

We can use JQuery to select the document object and add an event listener called ready. This event listener will fire as soon as the DOM is ready to be manipulated, but before all images are downloaded. Passing an anonymous function in ready () will execute our code when the document is ready for JavaScript manipulation.
Example: $(document).ready(function(){
alert("the document has finished loading!");
});

Commonly Used JQuery Selectors

JQuery selectors allow you to select and manipulate HTML elements

  • ID Selector $("#id")
  • Class Selector $(".class")
  • Element Selector ("p")
  • Multiple Selector $("p.intro") selects all paragraphs with a class of intro. ("h1,div,p") selects all h1, div, and p elements
  • All Selector $("*")
  • Attributes $("[href]") selects all elements with the href attribute
Commonly Used JQuery Event Methods
  • click, dblclick
  • mouseenter, mouseleave
  • keypress, keydown, keyup
  • submit, change, focus, blur
  • load, resize, scroll, unload

JQuery DOM Manipulation

Three common methods for DOM manipulation are:

  • text()
  • html()
  • val()

Input field:

  • $("#change1").text("This has been changed") will change the text of an element that has an id of change1
  • $("#change2").html("<strong>Change this HTML</strong>") will change the html of the element that has an id of change2
  • $("#change3").val("Goodbye") will change the value of the input field that has an id of change3

You could attach these methods to an event handler like a click event.
